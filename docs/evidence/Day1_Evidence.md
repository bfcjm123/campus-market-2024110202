三、遇到的问题与解决办法
问题：在安装依赖的时候遇到权限问题
解决：使用管理员权限执行安装命令，成功安装依赖。
问题：CMD 执行cd切换项目路径提示路径不存在
解决：Windows 跨磁盘跳转必须先输入盘符切换磁盘，再逐层进入文件夹，修正跳转步骤后成功进入项目根目录。
问题：执行git push -u origin main推送代码报错，提示找不到 main 分支
解决：查询本地分支名为master，更换推送命令git push -u origin master，代码推送成功。
